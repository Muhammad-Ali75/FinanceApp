## Prerequisites
* Node
* Yarn
* Expo Cli
* Expo Go App in mobile or emulator

## Installation

To run this just 

```bash
  yarn install
```
After that completes just run:    
```bash
  yarn start
```
then scan QR code from Expo Go
(in case of Iphone you need to scan QR from camera app)
